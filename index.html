Project Overview & Requirements

This software solution manages metadata for various media types, providing a persistent backend API and a user-friendly GUI.

System Architecture

Backend (Flask)

The backend handles data persistence (JSON/Pickle) and API requests.

Frontend (PyQt/PySide)

The frontend is a desktop GUI client communicating with the backend via HTTP/JSON.

List All Media, List by Category, Display Metadata.

Search by Name, Create, Update, Delete Media.
